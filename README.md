# About Blume

Blume is a widely compatible server-side plugin with many vanilla+ features, including:

- <img width="32" src="https://ccvaults.com/assets/10.%20Items/36.%20Other/Enchanted_Book.png" alt="Enchanted Book" align="center"> **Custom enchants** ex: veinminer, timber, unbreakable
- <img width="32" src="https://ccvaults.com/assets/10.%20Items/23.%20Flowers/wildflowers.png" alt="Cornflower" align="center"> **Ecological changes** ex: wild crop patches in some biomes
- <img width="32" src="https://ccvaults.com/assets/20.%20Blocks/39.%20Admin/Command_Block.png" alt="Command Block" align="center"> **Admin** ex: broadcasts every command executed in chat for everyone to see, graylist (modified from [Vane](https://github.com/oddlama/vane)): when new players join they need to be `/vouched <username>` by another user to break blocks, open doors, etc.; keeps logs of who vouched who

Refer to our [website](https://junerhobart.github.io/blume/) for a more comprehensive list.

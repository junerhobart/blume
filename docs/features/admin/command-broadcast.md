---
title: Command Broadcast
icon: minecraft:book
enabled_by_default: true
---

Cheat and admin commands show up in chat for everyone, vanilla (`/gamemode`, `/give`, `/fill`, `/tp`), Generic (`/fly`, `/god`, `/invsee`, `/vanish`), OpenInv, CMI, and WorldEdit (`/we`, `/brush`, any `//` command). Normal player commands like `/msg` and `/home` stay private. Edit `admin.command-broadcast.commands` in config to add or remove roots.
